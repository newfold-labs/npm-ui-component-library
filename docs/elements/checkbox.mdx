# Checkbox
The Checkbox element is used to select one or more options from a list of options.

<NewfoldRoot>
    <Checkbox
        id="checkbox"
        label="I am a checkbox."
        name="name"
        value="value"
    />
</NewfoldRoot>

---
### Usage
```jsx
import { Checkbox } from "@newfold/ui-component-library";

<Checkbox
    id="checkbox"
    label="I am a checkbox."
    name="name"
    value="value"
/>
```

---
### Disabled
No user interaction, indicated by opacity and cursor.

<NewfoldRoot>
    <Checkbox
        disabled
        id="checkbox-disabled"
        label="I am a checkbox."
        name="name"
        value="value"
    />
</NewfoldRoot>

```jsx
<Checkbox
    disabled
    id="checkbox-diabled"
    label="I am a checkbox."
    name="name"
    value="value"
/>
```
