import { Root as NewfoldRoot, AppBarNavigation, SidebarNavigation } from "@newfold/ui-component-library";
import { useNavigationContext } from "@newfold/ui-component-library";

# useNavigationContext

A hook that provides access to the navigation context. This hook is used to access navigation state and controls shared between `AppBarNavigation` and `SidebarNavigation` components.

## Usage

```jsx
import { useNavigationContext } from "@newfold/ui-component-library";

function MyComponent() {
    const { 
        activePath, 
        setActivePath, 
        isMobileMenuOpen, 
        setMobileMenuOpen 
    } = useNavigationContext();

    return (
        <div>
            <p>Current active path: {activePath}</p>
            <p>Mobile menu is {isMobileMenuOpen ? 'open' : 'closed'}</p>
            <button onClick={() => setActivePath('/new-path')}>
                Set Active Path
            </button>
            <button onClick={() => setMobileMenuOpen(!isMobileMenuOpen)}>
                Toggle Mobile Menu
            </button>
        </div>
    );
}
```

## Context Properties

The `useNavigationContext` hook returns an object with the following properties:

### `activePath`
- **Type**: `string`
- **Description**: The current active navigation path

### `setActivePath`
- **Type**: `function`
- **Description**: Function to update the active navigation path

### `isMobileMenuOpen`
- **Type**: `boolean`
- **Description**: Whether the mobile navigation menu is currently open

### `setMobileMenuOpen`
- **Type**: `function`
- **Description**: Function to control the mobile navigation menu open state

## Example with Navigation Components

<NewfoldRoot>
    <AppBarNavigation activePath="/dashboard">
        <AppBarNavigation.AppBar>
            <AppBarNavigation.Item
                label="Dashboard"
                href="/dashboard"
            />
            <AppBarNavigation.Item
                label="Settings"
                href="/settings"
            />
        </AppBarNavigation.AppBar>
    </AppBarNavigation>
</NewfoldRoot>

The `useNavigationContext` hook is automatically provided by the `NavigationProvider` context, which is used internally by both `AppBarNavigation` and `SidebarNavigation` components. This allows these components to share navigation state and work together seamlessly.
