# useRootContext
The `useRootContext` hook returns the root context.

### Usage
```jsx
import { useRootContext } from from "@newfold/ui-component-library";

const Component = () => {
    const { isRtl } = useRootContext();

    return <div className={ isRtl ? "nfd-ml-4" : "nfd-mr-4" } />
};
```
