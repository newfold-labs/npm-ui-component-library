# Notifications
The Notifications component shows notifications on a specified position on the screen. Switch position in the control panel to view.

<NewfoldRoot >
    <Notifications>
        <Notifications.Notification
            description="Notification description"
            dismissScreenReaderLabel="Dismiss"
            id="notification"
            title="Notification title"
            onDismiss={ (id) => { console.log('dismissed', id); } }
        />
    </Notifications>
    <p>Default notification position is bottom-left.</p>
</NewfoldRoot>

---
### Usage
```jsx
import { Notifications } from "@newfold/ui-component-library";

<Notifications>
    <Notifications.Notification
        description="Notification description"
        dismissScreenReaderLabel="Dismiss"
        id="notification"
        title="Notification title"
        onDismiss={ (id) => { console.log('dismissed', id); } }
    />
</Notifications>
```
