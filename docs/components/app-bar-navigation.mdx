# AppBar Navigation
An app bar navigation component for horizontal navigation. The component contains the [`AppBar`](#appbar), [`Item`](#item), and [`Mobile`](#mobile) sub-components and uses the [`NavigationProvider`](#navigationprovider) context for state management.

## NavigationProvider
The `AppBarNavigation` component uses the `NavigationProvider` context to manage navigation state. This context provides the following properties:

- `activePath` - The current active path.
- `setActivePath` - Function to set the active path.
- `isMobileMenuOpen` - A boolean indicating if the mobile menu is open.
- `setMobileMenuOpen` - A function to set the mobile menu open state.

The `NavigationProvider` is shared between `AppBarNavigation` and `SidebarNavigation` components, allowing them to work together seamlessly.
