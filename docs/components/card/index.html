Card

A simple card component. It has subcomponents for header, content and footer that has as and className props.

const Card = ();

This is Card header!
This is Card content!
This is Card footer!

Usage​

import { Card } from "@newfold/ui-component-library";

<Card>
    <Card.Header>
        This is Card header!
    </Card.Header>
    <Card.Content>
        This is Card content!
    </Card.Content>
    <Card.Footer>
        This is Card footer!
    </Card.Footer>
</Card>
