# Checkbox Group
The `CheckboxGroup` allows users to select one or more items from a list of choices.

<NewfoldRoot>
    <CheckboxGroupDemo values={['1', '3']}/>
</NewfoldRoot>

---
### Usage
```jsx
import { CheckboxGroup } from "@newfold/ui-component-library";

const options = [
    { label: 'Option 1', value: '1' },
    { label: 'Option 2', value: '2' },
    { label: 'Option 3', value: '3' },
    { label: 'Option 4', value: '4' }
];

<CheckboxGroup
    id="checkbox-group"
    label="A checkbox group"
    onChange={handleChange}
    options={options}
    values={['1', '3']}
/>
```

---
### Options
You can pass options to the `CheckboxGroup` component in two ways.
You can pass them as an array of objects to the `options` prop, or
you can pass them as children to the component.
<CheckboxGroupOptions />

---
### Label
Use the `label` prop to add a label to the `CheckboxGroup` component.

<NewfoldRoot>
    <CheckboxGroupDemo 
        values={['2', '3']}
        label="This is a label"
    />
</NewfoldRoot>

```jsx
<CheckboxGroup
    id="checkbox-group-label"
    onChange={handleChange}
    options={options}
    values={selectedValues}
    label="This is a label"
/>
```

### Description
Use the `description` prop to add a description to the `CheckboxGroup` component.

<NewfoldRoot>

    />

```jsx

    id="checkbox-group-label"
    onChange={handleChange}
    options={options}
    values={selectedValues}
    label="This is a label"

```
