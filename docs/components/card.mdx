# Card
A simple card component. It has subcomponents for header, content and footer that has `as` and `className` props.


<NewfoldRoot>
    <Card>
        <React.Fragment key=".0">
            <Card.Header>
                This is Card header!
            </Card.Header>
            <Card.Content>
                This is Card content!
            </Card.Content>
            <Card.Footer>
                This is Card footer!
            </Card.Footer>
        </React.Fragment>
    </Card>
</NewfoldRoot>

---
### Usage
```jsx
import { Card } from "@newfold/ui-component-library";

<Card>
    <React.Fragment key=".0">
        <Card.Header>
            This is Card header!
        </Card.Header>
        <Card.Content>
            This is Card content!
        </Card.Content>
        <Card.Footer>
            This is Card footer!
        </Card.Footer>
    </React.Fragment>
</Card>
```
