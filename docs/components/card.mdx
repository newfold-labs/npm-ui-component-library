# Card
The `Card` component is a generic container for organizing product-like UI elements.

<NewfoldRoot>
    <div className="nfd-flex">
        <Card className="nfd-w-[300px]">
            <Card.Header>
                This is Card header!
            </Card.Header>
            <Card.Content>
                This is Card content!
            </Card.Content>
            <Card.Footer>
                This is Card footer!
            </Card.Footer>
        </Card>
    </div>
</NewfoldRoot>

---
### Usage
```jsx
import { Card } from "@newfold/ui-component-library";

<div className="nfd-flex">
    <Card className="nfd-w-[300px]">
        <Card.Header>
            This is Card header!
        </Card.Header>
        <Card.Content>
            This is Card content!
        </Card.Content>
        <Card.Footer>
            This is Card footer!
        </Card.Footer>
    </Card>
</div>
```

---
### Sub-Components
The `Card` component has three sub-components: `Card.Header`, `Card.Content`, and `Card.Footer`.

The `Card` component is very flexible and can be used in a variety of ways.
The example below shows how to use the card component without a header by
simply not including the `Card.Header` sub-component.

<NewfoldRoot>
    <div className="nfd-flex">
        <Card className="nfd-w-[300px]">
            <Card.Content>
                This is Card content!
            </Card.Content>
            <Card.Footer>
                This is Card footer!
            </Card.Footer>
        </Card>
    </div>
</NewfoldRoot>

```jsx
<div className="nfd-flex">
    <Card className="nfd-w-[300px]">
        <Card.Content>
            This is Card content!
        </Card.Content>
        <Card.Footer>
            This is Card footer!
        </Card.Footer>
    </Card>
</div>
```

You can do the same with the `Card.Content` and `Card.Footer`.
